# VS Code Extension — Install Guide

## Quick Install

1. Download [RA.vsix](RA.vsix)
2. Open VS Code
3. `Ctrl+Shift+P` → `Extensions: Install from VSIX...`
4. Select `RA.vsix`

## Verify

Open any `.ra` file — syntax highlighting activates automatically.

## Uninstall

1. VS Code Extensions panel
2. Find "RA Language"
3. Click "Uninstall"
