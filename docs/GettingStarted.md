# Getting Started with RA

RA is a modern programming language designed for simplicity and power.

## Quick Start

1. **Download** the latest release from [downloads](../downloads/)
2. **Install** using the installer or portable package
3. **Run** your first program:

```bash
ra run examples/HelloWorld.ra
```

## Prerequisites

- Windows 10/11, Linux, or macOS
- No runtime dependencies required
