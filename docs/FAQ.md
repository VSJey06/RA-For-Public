# Frequently Asked Questions

### What is RA?
RA is a modern programming language focused on readability, safety, and developer experience.

### Is RA free?
Yes. RA is open source under the MIT license.

### What platforms are supported?
Windows 10/11, Linux (x86_64), and macOS (x86_64, ARM).

### Do I need to install Python?
No. RA executables are self-contained. Python is only needed for development of the language itself.

### How do I report a bug?
Open an issue at https://github.com/RA-Lang/RA/issues

### How do I contribute?
See the [CONTRIBUTING.md](../CONTRIBUTING.md) guide.

### Where are language docs?
The full specification is available in the private development repository. This public repo contains user-focused documentation only.
