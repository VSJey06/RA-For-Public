# Release Notes

## v1.0.0 — Initial Public Release

### Features
- RA language interpreter
- RADK development toolkit
- RAPM package manager
- VS Code extension (syntax highlighting, LSP, debugger)
- Portable mode (USB-ready, no install required)
- Project templates (console-app, ai-demo, dsa-demo, library)
- Package registry with trust model

- [Windows Installer](installers/RA_Setup.exe)
- [Portable Build](downloads/RA_Portable/)
- [Release Binaries](downloads/Release/)
- [VS Code Extension](vscode/RA.vsix)

### Installation
```bash
# Windows (PowerShell)
powershell -ExecutionPolicy Bypass -File install/install.ps1

# Linux/macOS
chmod +x install/install.sh && ./install/install.sh
```
