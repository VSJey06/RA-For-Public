# Changelog

All notable changes to the RA Language project are documented in this file.

## [1.0.3] - 2025

### Added
- PF (Program Framework)
- Program Handler blocks
- Function Flow blocks
- Check and Key validation

## [1.0.2] - 2025

### Added
- OOP library
- Constructors
- Encapsulation
- Object methods
- Method dispatch
- `.run` blocks
- `.fun` blocks
- Runtime commands
- Float support in I datatype
- Numeric suffix support (K, Lh, Cr, B, Tri, Qd)

## [1.0.1] - 2025

### Added
- Database blocks
- Database save
- Database load
- Property assignment

## [1.0.0] - 2025

### Added
- String datatype (S)
- Numeric datatype (I)
- Print
- Classes
- Objects
- Methods
- If statements
- For loops
- While loops
